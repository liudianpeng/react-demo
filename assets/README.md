# 前端项目

> 基于 react redux antd webpack es6

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

```
## 项目特点
- 前后端分离，前端dev-server反向代理请求

## 文件命名规范
jax文件名首字母大写
js文件名首字母小写
